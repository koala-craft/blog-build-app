# 実行環境構築手順書 (自動化版)

> 本書は `setup.py` を使用して、バックエンド（FastAPI）とフロントエンド（Next.js）の環境を最短で構築するための手順書です。

---

## STEP 0: 事前準備

以下がインストール済みであることを確認してください。
- **Python 3.11+** (`python --version`)
- **Node.js 18+** (`node --version`)
- **Docker Desktop** (`docker --version`)

---

## STEP 1: プロジェクトの初期化 (一括生成)

プロジェクトのルートディレクトリで以下のコマンドを実行します。
これ一回で、フォルダ構成・設定ファイル・DB初期化スクリプトがすべて作成されます。

```bash
python setup.py
```

---

## STEP 2: データベースの起動

```bash
docker compose up -d
```

> **自動化のポイント:** 起動時に `init-db/01-vector.sql` が自動実行され、`pgvector` 拡張が有効化されます。手動でのSQL実行は不要です。

---

## STEP 3: バックエンド (FastAPI) のセットアップ

1. **仮想環境の作成と起動**
   ```bash
   cd backend
   python -m venv .venv
   # Windowsの場合
   .venv\Scripts\activate
   # Mac/Linuxの場合
   source .venv/bin/activate
   ```

2. **依存パッケージのインストール**
   ```bash
   pip install -r requirements.txt
   ```

3. **APIキーの設定**
   `backend/.env` を開き、あなたの `OPENAI_API_KEY` と `ANTHROPIC_API_KEY` を記入してください。

4. **動作確認**
   ```bash
   uvicorn app.main:app --reload --port 8000
   ```
   `http://localhost:8000/docs` にアクセスできれば成功です。

---

## STEP 4: フロントエンド (Next.js) のセットアップ

1. **Next.js プロジェクトの作成**
   ```bash
   cd ../frontend
   npx create-next-app@latest . --pnpm --typescript --tailwind --eslint --app --src-dir --no-import-alias
   ```
   *(すべて `Enter` (Yes) で進めてOKです)*

2. **追加ライブラリのインストール**
   ```bash
   pnpm add lucide-react react-markdown swr
   ```

3. **動作確認**
   ```bash
   pnpm run dev
   ```
   `http://localhost:3000` にアクセスできれば成功です。

---

## まとめ：これだけで準備完了です

以後は以下のコマンドで開発を継続できます。
- **DB起動**: `docker compose up -d`
- **バックエンド**: `cd backend && uvicorn app.main:app --reload`
- **フロントエンド**: `cd frontend && npm run dev`

> 準備ができたら「SQLAlchemy モデルの実装を開始して」と声をかけてください。
